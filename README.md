# HR Workforce Analytics Dashboard

## Project Overview
This project is an HR Workforce Analytics Dashboard developed using Microsoft Power BI. It provides insights into employee attrition, workforce distribution, job satisfaction, and other HR metrics to support data-driven decision-making.

## Tools Used
- Microsoft Power BI
- Microsoft Excel / CSV

## Dashboard KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Active Employees
- Average Monthly Income
- Average Job Satisfaction
- Average Years at Company

## Visualizations
- Attrition by Department
- Employee by Age Group
- Job Satisfaction by Role
- Attrition by Education Field
- Work-Life Balance Distribution
- Employee Distribution by Job Role
